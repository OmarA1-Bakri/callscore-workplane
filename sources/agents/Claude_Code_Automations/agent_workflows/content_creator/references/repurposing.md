## 1. Core Principle

One idea, many formats. Create once, adapt everywhere.

The adaptation is NOT copy-paste. Each platform has different:
- Audience expectations
- Format constraints
- Engagement patterns
- Algorithm preferences

## 2. Blog Post to Multi-Platform

Starting material: 800-2000 word blog post.

### Blog --> LinkedIn Post
1. Extract the single strongest insight
2. Write a 2-line hook (55 chars each)
3. Distill to 3-5 short paragraphs
4. End with a specific question as CTA
5. Create a 1080x1350 image summarizing the key point
6. Put the blog link in the first COMMENT, not the post body

### Blog --> LinkedIn Carousel
1. Extract 5-8 key insights
2. Create one slide per insight
3. Add a hook slide (Slide 1) and CTA slide (last)
4. Design in Gamma.app, export as PDF
5. Post with a 2-line hook and brief body text

### Blog --> X/Twitter Thread
1. Distill blog to 5-8 key points
2. Write each as a standalone tweet (280 chars)
3. Number them 1/ through N/
4. First tweet = strongest hook from the blog
5. Last tweet = CTA with link to full article
6. Post via Rube (TWITTER_CREATION_OF_A_POST)

### Blog --> X/Twitter Single Tweet
1. Find the one most tweetable line from the blog
2. Sharpen it to under 280 chars
3. Optionally add the blog link
4. Post via Rube (TWITTER_CREATION_OF_A_POST)

### Blog --> Newsletter Teaser
1. Write 2-3 sentence teaser (the "cliffhanger")
2. Include one key insight as a taste
3. Link to full blog post
4. Add a personal note about why you wrote it

## 3. LinkedIn Post to Other Platforms

### LinkedIn Post --> Tweet
1. Take the hook (2 lines)
2. Compress to 280 chars or less
3. Keep the core tension or insight
4. Drop the CTA (tweets are more conversational)

### LinkedIn Post --> Thread
1. Take the full LinkedIn post body
2. Break into 3-5 tweet-sized chunks
3. Add numbering (1/, 2/, etc.)
4. First tweet = LinkedIn hook compressed

### LinkedIn Post --> Blog Post
1. Expand each paragraph into a full section
2. Add data, examples, and deeper analysis
3. Structure with H2 headings
4. Add SEO keywords in title and subheadings
5. Include images or diagrams

---

## 4. Tweet/Thread to Other Platforms

### Thread --> LinkedIn Post
1. Combine the thread into flowing paragraphs
2. Write a new 2-line hook (LinkedIn style)
3. Expand key points with more detail
4. Add a 1080x1350 image
5. End with a question CTA

### Thread --> Blog Post
1. Each tweet becomes a section outline
2. Expand each into 150-300 words
3. Add connecting narrative between sections
4. Include supporting evidence and examples
5. Optimize for SEO

### Tweet --> Newsletter Mention
1. Embed the tweet insight in your newsletter
2. Add personal context ("I posted this and...")
3. Share the response/engagement data
4. Connect to the newsletter's main theme

## 5. Cross-Platform Adaptation Rules

| Rule | LinkedIn | X/Twitter | Blog | Newsletter |
|------|----------|-----------|------|------------|
| Tone | Professional authority | Casual, punchy | Thorough, structured | Conversational |
| Length | 200-300 words | 280 chars / 3-10 tweets | 800-2000 words | 500-1200 words |
| Links | In first comment only | In tweet body OK | Inline | Inline |
| Hashtags | Never | Rarely | In metadata/SEO | Never |
| Emojis | Never in body | Sparingly OK | Never | Sparingly OK |
| Images | 1080x1350, scrappy | 1200x675, clean | Featured + inline | Optional |
| CTA | Specific question | Follow/retweet/link | Next action | Reply/forward |
| Hook style | 2-line open loop | One punchy line | Title + first paragraph | Subject line |
