## 3. Brand Voice

### Tone Attributes
- **Conversational authority**: Knows the subject deeply but explains like a friend
- **Direct**: No hedging, no "it could be argued that..."
- **Honest**: Shares real numbers, real failures, real opinions
- **Reader-focused**: Every sentence serves the reader, not the ego

### Voice Do's
- Use "you" more than "I"
- Share specific numbers and examples
- Take clear positions (not "it depends")
- Write like you're texting a smart friend
- Use short sentences for emphasis. Like this.

### Voice Don'ts
- Never start with "In today's..." or "In the ever-evolving..."
- Never use "leverage", "synergy", "paradigm shift", "landscape"
- Never hedge with "It's worth noting" or "It's important to mention"
- Never use emojis in LinkedIn post copy
- Never use hashtags
- Never write about yourself unless it directly serves the reader

## 4. Writing Rules

These rules apply across ALL platforms:

1. **Reader first**: Every post is about the reader's problem, never the creator's achievements
2. **One idea per piece**: Don't try to cover everything. Go deep on one thing.
3. **Specific over general**: "Grew followers by 1,247 in 12 days" beats "grew my audience quickly"
4. **Show don't tell**: Use examples, stories, and data instead of assertions
5. **Earn every word**: If a sentence doesn't add value, cut it
6. **Strong openings**: Hook in the first 2 lines (LinkedIn) or first sentence (X/blog)
7. **Clear CTA**: Every piece should make the reader want to do something specific
8. **No AI slop**: Run everything through humanizer before publishing

### Topic-to-Lane Mapping

Each pillar naturally maps to 1-2 ICP lanes. When drafting, pick the pillar first, then check which lane it serves. This ensures content rotation covers all 5 audiences over time.

### Evergreen vs Timely Split
- **70% evergreen**: Infrastructure explainers, frameworks, how-it-works posts (long shelf life, compounds over time)
- **30% timely**: Regulatory news reactions, conference takeaways, competitor moves, market data (engagement spikes, short shelf life)
