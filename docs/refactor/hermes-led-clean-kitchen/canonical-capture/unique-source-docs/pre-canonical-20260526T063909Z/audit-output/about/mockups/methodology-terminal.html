#METHODOLOGY

How we score every call.

We run every public crypto call against 18.7 million candles. No black boxes. Every formula, every weight, every data source—laid out right here.

//how rankings work

$pipeline --describe ├── scrape fetch transcripts from 20 creators, daily ├── extract LLM-identifies buy/sell calls with ticker + direction ├── match align each call to the 30-day candle window that follows ├── score compute return_30d vs BTC benchmark └── rank Wilson-adjusted Alpha Score with Low-N penalty // confidence floor: extraction threshold > 70% // update cadence: daily, 00:00 UTC

03 / components

//score components

// direction — binary — 40 pts max
direction_correct ......................................... 40

Did the price go the direction they called at 30 days? Bullish call + price up = 40. Wrong direction = 0.

// alpha_over_btc — capped — 25 pts max
alpha_30d ......................................... 25

How much did the coin outperform Bitcoin over 30 days? Each 1% of alpha = 2.5 pts, capped at 25.

// specificity — composite — 15 pts max
specificity_score ......................................... 15

How precise was the call? entry (3.75) · target (3.75) · stop-loss (3.75) · timeframe (3.75).

// regime_difficulty — contextual — 10 pts max
regime_difficulty ......................................... 10

How hard was this call given market conditions? Bullish in a bear market = 10. Bullish in a bull market = 1.

// target_hit — binary — 10 pts max
hit_target_90d ......................................... 10

Did the price actually hit their stated target within 90 days? Yes = 10. No = 0.

06 / low-N adjustment

//wilson score & the small-sample problem

$cat wilson-adjustment.md // Low-N threshold = calls < 50 // Penalty = Wilson score lower bound (95% CI) // Why: a creator with 2 wins out of 3 has less statistical confidence // than a creator with 30 wins out of 50, even if raw win-rate is higher. wilson_lb = (p + z²/2n - z·√(p(1-p)/n + z²/4n²)) / (1 + z²/n) where p = win_rate, n = sample size, z = 1.96
