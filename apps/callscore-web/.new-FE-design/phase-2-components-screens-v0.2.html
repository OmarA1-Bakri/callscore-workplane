These primitives are where the product expression lives. Each is specified once, then deployed consistently across the screens. Best-call · editorial, not performance-color

before / after
accent is reserved for editorial highlight

The v0.1 "Best call" metric rendered the figure in --accent — the same treatment used for Alpha Score and editorial spots. This created ambiguity: readers might parse it as "accent = good = positive P/L," polluting the pos/neg semantic. In v0.2, accent is stripped off the figure and moved to a small editorial pill ("BEST"); the figure itself renders in --pos (since this particular value happens to be positive) with an explicit "+" sign and "α" unit, matching all other performance figures exactly.
